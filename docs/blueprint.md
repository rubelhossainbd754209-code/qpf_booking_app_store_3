# **App Name**: Repair Request Pro

## Core Features:

- Device Type: Device Type Selection: Allow customers to select their device type (e.g., mobile, laptop, tablet).
- Brand Selection: Brand Selection: Display brands based on the selected device type (e.g., Samsung, Apple, Xiaomi).
- Model Selection: Model Selection: Allow customers to select a specific model from the chosen brand.
- Issue Description: Issue Description: Provide a text box for customers to describe their device's issue in detail.
- Customer Information: Customer Information: Collect customer's name, mobile number, email address, and home address.
- Privacy Policy: Privacy Policy Agreement: Include a mandatory privacy policy checkbox.
- Request Submission: Request Submission: Enable customers to submit their repair request via a 'Booking' button.

## Style Guidelines:

- Primary color: Light desaturated blue (#B4D9FF) to communicate trust and dependability.
- Background color: Very light desaturated blue (#F0F8FF).
- Accent color: Light purple (#D8B4FF) to draw attention to calls to action.
- Body and headline font: 'PT Sans' (sans-serif) for a clean and modern user experience.
- Use consistent, professional icons related to device types and repair services.
- Design a clean, user-friendly layout with clear sections for each step of the request process.
- Use subtle animations for form transitions and feedback to improve user engagement.
